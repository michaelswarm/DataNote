# Sample SwiftData Notes App

## Prerequisites
- Xcode 15 or later
- iOS 17 or later

## Versions  
- 0.1.0 Base feature set: add, delete, select
- 0.2.0 Sort, modified 
- 0.3.0 Bulk operations: export, import, delete all 
- 0.4.0 Search: title search, content search 
- 0.5.0 Wiki
